Les liens

Pour inserer des liens sur une page, on utilise la balise < a href="(LeLien)" > (Nom du lien) < /a > : C'est un lien

Les images

Pour inserer des images sur une page, on utilise la balise < img src="(FichierImage)" alt="(TitreSiProbleme)" width="(largeur)" height="(longueur)"> :

Les vidéos

Pour inserer une vidéo sur une page, on utilise la balise

< video width="(largeur)" height="(hauteur)" (controls ou autoplay)>
< source src="(FichierVideo)" type="video/(formatdelavideo)>
< /video >

Attention si on met autoplay il peut arriver que la vidéo ne se lance pas car les navigateurs l'ont bloqué pour des raisons de sécurité, en ajoutant muted derriere elle se lancera mais sans son.
